# Security Policy

OpenPlanTrace is early-stage software and should be treated as an analysis engine, not a security boundary.

## Reporting

Please report security issues privately to the repository owner instead of opening a public issue with exploit details.

## Sensitive Test Data

Do not attach or commit private architectural drawings, customer PDFs, DWGs, screenshots, model weights, API keys, or local benchmark outputs. Use local-only ignored paths for private testing.

## Supported Versions

Security fixes are currently handled on the `main` branch until versioned releases exist.
